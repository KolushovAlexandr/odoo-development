oe_dark
=======

Use ``oe_dark`` class on every even ``section``. Don't use ``oe_dark`` for beginning and ending sections.

.. code-block:: html

    <section class="oe_container">
        <!--Title-->
    </section>
    
    <section class="oe_container">
        <!--Key features-->
    </section>

    <section class="oe_container">
    </section>

    <section class="oe_container oe_dark">
    </section>

    <section class="oe_container">
    </section>

    </section>
